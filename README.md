# TS-Maze-Game

A small game in which one has to find the goal in various mazes, while dodging or fighting enemies.

## Installation

You can install and play this Game on your PC. Here are the main steps:

* Download and install [Node.js](https://nodejs.org) v24 or higher.
* Download the latest Version of the code.
* Install the dependencies of the project by running `npm install` in the project folder.
* Run `npm start`.
* Now you can play!

## Tutorial

* Press W/S or arrow keys for menu selection
* Space for confirming the selection in menus
* ? for in-game help (keybind list)

## Instruction

* First you can play the three tutorial levels one after the other.
* After that, levels will be unlocked. More levels can be unlocked by completing previous ones.
* In freeplay mode, an infinite number of mazes are created randomly.
